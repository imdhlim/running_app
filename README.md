# 🏃‍♂️ Hodadak 
위치기반 코스추천 러닝 앱

---

## 📌 프로젝트 개요
러닝 앱 "호다닥"은 기존의 러닝 앱과는 다른<br>
사용자들이 직접 경험한 코스를 공유하여<br>
어디서든 쉽게 러닝을 할 수 있도록 도움을 주고<br>
지역 활성화와 동기부여를 위한 추천 코스 기반 러닝 앱

---

## 🎯 기획 의도
- 혼자 러닝할 때 **어디를 달려야 할지 모르는 문제**
- 여행지나 새로운 동네에서 **검증된 러닝 코스 부족**
- 단순 기록 앱이 아닌 **사용자 참여형 러닝 플랫폼** 필요

➡️ 실제 사용자가 달린 코스를 기반으로  
➡️ 추천 수에 따라 신뢰도 있는 경로 제공

---

## 🛠 주요 기능

### 1️⃣ 실시간 러닝 트래킹
- GPS 기반 실시간 위치 추적
- 운동 시간, 이동 거리, 평균 페이스 계산
- 칼로리 소모량 자동 계산
- 실시간 데이터 업데이트

### 2️⃣ 운동 데이터 관리
- 러닝 종료 시 운동 데이터 저장
- 운동 요약 화면 제공
- 캘린더를 통한 날짜별 운동 기록 관리

### 3️⃣ 코스 공유 기능
- 게시판에 직접 달린 러닝/산책 경로 공유
- 사진 및 설명 추가 가능

### 4️⃣ 코스 추천 게시판
- 추천(좋아요) 수 기반 정렬 가능
- 내 위치 기반 정렬 가능

### 5️⃣ 랭킹 시스템
- 한 달 누적 러닝 거리 순 통합 랭킹
- 초급자, 중급자, 상급자 별 랭킹
